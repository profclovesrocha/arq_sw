# Sistema de Apoio à Alfabetização Hospitalar

Projeto acadêmico desenvolvido para disciplina de Arquitetura de Software.

## Descrição
Sistema web para apoio à alfabetização em ambiente hospitalar com IA generativa simulada.

## Arquitetura
Modelo em Camadas:
- Apresentação
- Controle
- Regra de Negócio
- Dados

## Tecnologias
- Python
- Flask
- HTML/CSS

- templates/
